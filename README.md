# WB-MIL
A prototype program for automize Polish Military Police (Military Gendarmerie) work.
- Bases on XML structure of Word file.
- Currently there is only one theme.
- Generates .pdf or .doc file.

### Used libraries
-  DocumentFormat.OpenXml.Packaging;
- DocumentFormat.OpenXml.Wordprocessing;
-  Syncfusion.DocIO;
-  Syncfusion.DocIO.DLS;
-  Syncfusion.DocToPDFConverter;
-  Syncfusion.Pdf;
